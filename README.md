# Flutter GetX Projects Collection

A collection of Flutter projects built using **GetX** to help understand state management, dependency injection, routing, and the MVC architecture.

This repository is intended for beginners and students who want to learn GetX through practical examples.

## 📚 Repository Contents

This repository includes multiple GetX-based projects such as:

* Counter App
* Todo App
* CRUD Operations
* Product Management
* API Integration
* Timer App
* Navigation Examples
* Form Validation
* Authentication Examples
* More projects will be added regularly.

## 🎯 Objectives

* Learn GetX State Management
* Understand MVC Architecture
* Practice Dependency Injection using Bindings
* Learn Named Routing
* Build scalable Flutter applications
* Understand reactive programming using `Obx()`

## 📂 Repository Structure

```text
Flutter_GetX_Projects/
│
├── Counter_Timer_App/
├── Todo_App/
├── Product_CRUD/
```

Each project is self-contained and can be opened independently in Flutter.

## 🚀 Getting Started

1. Clone the repository.

```bash
git clone https://github.com/your-username/flutter-getx-projects.git
```

2. Open any project folder.

3. Install dependencies.

```bash
flutter pub get
```

4. Run the project.

```bash
flutter run
```
